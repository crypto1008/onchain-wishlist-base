# Scripts

## deploy.js
Deploys WishList contract to Base Mainnet.
Run: npx hardhat run scripts/deploy.js --network base

## verify.js
Verifies contract on Basescan.
Run: npx hardhat run scripts/verify.js --network base

## interact.js
Reads contract data from Base Mainnet.
Run: npx hardhat run scripts/interact.js --network base
